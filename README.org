* Эволюционно-генетический алгоритм

Решение поделено на 4 проекта:
- ~Common~. Проект, задающий общие для всех проектов методы и классы.
- ~EA~. Проект, задающий "скелет" эволюционно-генетического алгоритма.
- ~TSP~. Проект, задающий задачу коммивояжёра в контексте ЭГА.
- ~App~. Исполняемый проект.
